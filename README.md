# YuGangBot
  
一个基于开源框架[maria](https://github.com/mamoe/mirai)实现的娱乐性机器人

## 声明
本项目仅用与个人学习/娱乐使用，不鼓励，不支持一切商业使用。
  
本项目采用`AGPlv3`协议开源
